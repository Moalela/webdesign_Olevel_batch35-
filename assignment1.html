Q)Define Web browsers ,web site,web page and webserver Ans:1)A web browser can be defined as an application for accessing to certain websites.Example like Chrome,Internet explorer,Mozilla firefox etc. 2)A website can also be termed as webpages with some display on the mainscreen for the viewers example while opening instagram page ,facebook page etc. 3)A webpage is a page written in hypertext, webpages mostly includes videos and images and also a group of other web pages while tapping on it. 4)A webserver stores the data that a client needs as well as serving of website files.
